# Online Quiz Web Application

## 📌 Project Description
This is a simple online quiz web application where users can create their own quiz questions, attempt the quiz, and view their results.

## 🚀 Features
- User Login Module
- Quiz Creation Module (Add custom questions)
- Quiz Attempt Module
- Quiz Result Module

## 🛠️ Technologies Used
- HTML
- LocalStorage (for data storage)

## 📂 Project Modules
1. Login Module – User enters name
2. Quiz Creation Module – Create questions and answers
3. Quiz Attempt Module – Attempt quiz
4. Result Module – Display score

## ▶️ How to Run
1. Open `index.html`
2. Login and create questions
3. Start quiz and answer questions
4. View your result

## 📸 Output
- Displays score like: `2 / 3`
